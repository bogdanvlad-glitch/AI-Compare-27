# Portfolio 5 — L'Esplanade Louvain-la-Neuve

Site vitrine statique multi-pages consacré à L'Esplanade de Louvain-la-Neuve.

## Pages

- `index.html` : page d'accueil visuelle
- `decouvrir.html` : boutiques, restaurants, catégories et services
- `ambiance.html` : atmosphère, ville piétonne, Grand-Place, gare et lac
- `galerie.html` : nouvelle galerie avec plus d'images
- `infos-pratiques.html` : horaires, adresse, accès et conseils

## Améliorations ajoutées

- Nouveau design premium avec header glassmorphism, cartes arrondies, ombres et grille responsive
- Plus d'images sur toutes les pages
- Nouvelle page Galerie
- Animations au scroll et navigation mobile conservées
- Corrections d'enseignes et de catégories basées sur les pages publiques consultées le 29 avril 2026
